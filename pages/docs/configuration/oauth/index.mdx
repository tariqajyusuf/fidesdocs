# API Authentication with OAuth

The Fides API requires authentication via OAuth using a valid Access Token. In this section we'll explore how to generate a token, create additional OAuth clients and authenticate with appropriate scopes to the Fides API.
If you're looking for information on configuring a root OAuth client, read the [Root Client Configuration](#LINK) guide here.

For details, see:

* [API Authentication](api-authentication)
* [OAuth Token Expiration](token-expiration)

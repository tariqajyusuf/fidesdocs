# Fides 2.19.1/2/3 Release notes
September 12-15, 2023

These release notes are for Fides & Fidesplus 2.19.1, 2.19.2, 2.19.2

### Features and Enhancements
- System history UI with a modal to examine the details of changes

### Fixes and developer updates
- Fixed bug with how data map was handling data use declarations without Data Subjects
- Re-enabled custom fields for data use declarations
- Fixed issue with saving source and destination systems
- Legal basis for transfer now takes any string as input
- Ensured dictionary and job settings subsections are initialized in cases of where there is no `.toml` configuration file

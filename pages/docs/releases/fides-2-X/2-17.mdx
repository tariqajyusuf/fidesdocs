# Fides 2.17 Release notes
July 25, 2023

## Fides for Business 2.17.0
### Enhancements
- **Fixes**
    - Privacy notice UI's list of possible regions is now correct. 



## Fides OSS 2.17.0
### Enhancements
- **Enable/disable taxonomy elements**: a toggle has been added to taxonomy elements like data uses to allow admins to disable these elements so that they don't appear in the picker [#3844](https://github.com/ethyca/fides/pull/3844).
- **Updated integration templates**: integration templates now accurately show required vs optional fields and maintain the visibility of entered values after save. Password fields are now masked on entry [#3775](https://github.com/ethyca/fides/pull/3775), [#3843](https://github.com/ethyca/fides/pull/3843)
- **Fixes**
    - Broken link to Fides docs site on the About Fides page in Admin UI [#3643](https://github.com/ethyca/fides/pull/3643)
    - Fixed issue preventing ConnectionConfigs with duplicate names from saving [#3770](https://github.com/ethyca/fides/pull/3770)
    - Fixed creating and editing manual integrations [#3772](https://github.com/ethyca/fides/pull/3772)
    - Fix lingering integration artifacts by cascading deletes from System [#3771](https://github.com/ethyca/fides/pull/3771)
    - System cards are now sorted alphabetically by name on "View systems" page [#3781](https://github.com/ethyca/fides/pull/3781)
    - Pinned pymssql and cython dependencies to avoid build issues on ARM machines [#3829](https://github.com/ethyca/fides/pull/3829)
    - Added an active property to taxonomy elements [#3784](https://github.com/ethyca/fides/pull/3784)
    - Admin UI "property does not existing" build issue [#3831](https://github.com/ethyca/fides/pull/3831)
    - Added toast for successfully linking an existing integration to a system [#3826](https://github.com/ethyca/fides/pull/3826)
- **Developer experience Enhancements**:
    - Changed where db-dependent routers were imported to avoid dependency issues [#3741](https://github.com/ethyca/fides/pull/3741)
- **Updated versions**:
    - Bumped supported Python versions to 3.10.12, 3.9.17, and 3.8.17 [#3733](https://github.com/ethyca/fides/pull/3733)

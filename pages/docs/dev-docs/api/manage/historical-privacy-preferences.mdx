# API: historical-privacy-preferences

The `/api/v1/historical-privacy-preferences` is designed to give you a report of _historical_ consent preferences made by end-user subjects. To see _current_ preferences, please refer to the [current-privacy-preferences](/dev-docs/api/manage/historical-privacy-preferences) endpoint. 

This can be used to integrate into popular dashboard and reporting tooling.

# API: data_category

The `/api/v1/data_category` routes are used to create, read, update, and delete the [Data Categories](/tutorials/data-mapping-glossary#data-category) within the Fides platform. Data Categories can be customized but we recommend you use the [Fideslang](https://ethyca.github.io/fideslang/explorer/) taxonomy.

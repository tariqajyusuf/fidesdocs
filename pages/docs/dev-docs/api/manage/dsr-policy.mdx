# API: dsr/policy

The `/api/v1/dsr/policy` routes are a collection of routes that are used to [configure Privacy Requests](/dev-docs/configuration/privacy-requests). It is important to configure this correctly to properly fulfill [graph traversal](/dev-docs/configuration/privacy-requests/graphs).

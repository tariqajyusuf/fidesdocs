# API: evaluation

The `/api/v1/evaluation` routes are a collection of routes that are used to create, read, update, and delete [Fides evaluations](/dev-docs/cli/check/cmd-evaluate). This can be used to integrate with popular dashboard and reporting tools to get an overview on repo-level Privacy evaluations within your organization.

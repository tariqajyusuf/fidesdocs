# API: config

The `/api/v1/config` routes are a collection of routes that are used to create, update, and delete Fides deployment configuration.

This will include the existing runtime configuration such as CORS origins, Fides admin analytics, Fides database configurations, email/sms notification configurations, redis configurations, and Privacy Request execution configurations.

# API: dsr/policy/\{policy_key}/webhook

The `/api/v1/dsr/policy/\{policy_key}/webhook` routes are a collection of routes that are used to [configure Privacy Requests Webhooks](/dev-docs/configuration/privacy-requests/webhooks). This is a powerful tool that allows you to enrich identities (or handle pre/post events) when performing [graph traversal](/dev-docs/configuration/privacy-requests/graphs).

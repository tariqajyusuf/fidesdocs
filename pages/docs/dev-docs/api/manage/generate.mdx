# API: generate

The `/api/v1/generate` routes is a route that performs schema discovery on a Database and generates an _unclassified_ dataset. To generate a classified dataset, you need to have access to the enterprise classifier. If you need access to the classification endpoint, please reach out to your Customer Success Manager or [reach out to us for your enterprise needs](https://ethyca.com/book-demo?utm_source=docsiteapidocs).

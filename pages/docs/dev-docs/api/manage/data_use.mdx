# API: data_use

The `/api/v1/data_use` routes are used to to create, read, update, and delete the [Data Uses](/docs/tutorials/data-mapping-glossary#data-use) within the Fides platform. Data Categories can be customized but we recommend you use the [Fideslang](https://ethyca.github.io/fideslang/explorer/) taxonomy.

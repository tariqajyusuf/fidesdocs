# API: privacy-experience

The `/api/v1/privacy-experience` routes is a route that returns a list of [Privacy Experience](/tutorials/consent-management-fundamentals/consent-management-privacy-experiences) records for individual regions. You can use this endpoint to build a custom privacy experience using our APIs.

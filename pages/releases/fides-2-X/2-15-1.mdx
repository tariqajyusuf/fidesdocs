# Fides 2.15 Release notes
July 5, 2023 


## Fides OSS 2.15.0
### Enhancements
- **SSL mode for SSH**: `sslmode` is now set to `prefer` for SSH Connections [#3685](https://github.com/ethyca/fides/pull/3685).
- **Updated integration tab permissions**: fixed issue where permissions weren't being assessed correctly on the Integrations tab for systems [#3707](https://github.com/ethyca/fides/pull/3707).
- **DSR processing issue**: fixed issue where privacy requests were failing when encountering fields with a double underscore `_` [#3686](https://github.com/ethyca/fides/pull/3688)
- **Privacy notice scrolling**: fixed issue with scrolling in privacy banner and modal [#3713](https://github.com/ethyca/fides/pull/3713).
- **Resolved critical security vulnerability**: Resolved security vulnerability concerning path traversal in webserver API [CVE-2023-36827](https://github.com/ethyca/fides/security/advisories/GHSA-r25m-cr6v-p9hq)

# Getting Started with Privacy Requests

<TutorialKey duration="40" product="Fides" topic="Privacy Requests" interactive="true"/>

In this series of tutorials, you'll configure integrations for processing privacy requests for your systems and walk through the steps to test a privacy request to check for issues that may occur in the request process.

By the end of this series, you'll be able to add privacy   request integrations to your systems, review and process privacy requests, and investigate issues if they occur.

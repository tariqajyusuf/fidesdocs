# Yotpo Loyalty
[Yotpo Loyalty](https://www.yotpo.com/platform/loyalty/) is an analytics platform that automatically captures and measures user interactions on websites and mobile apps for data-driven insights.

## Prerequisites

In order to [integrate](/user-guides/integrations) with Yotpo Loyalty, you'll need to collect the following information:

| Name                       | Description                                                                                                                             |
| :------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **Domain***                | The URL for your Yotpo Loyalty instance (default: `loyalty.yotpo.com` |
| **Merchant ID***          | Your Yotpo Merchant ID. |
| **GUID***                 | Your Yotpo Global Unique Identifier (GUID). Follow Yotpo Loyalty's docs for [steps to retrieve this information](https://support.yotpo.com/docs/finding-your-loyalty-referrals-api-key-and-guid) |
| **API key***                 | Your Yotpo API key. Follow Yotpo Loyalty's docs for [steps to retrieve this information](https://support.yotpo.com/docs/finding-your-loyalty-referrals-api-key-and-guid) |
| **Store ID***                 | Your Yotpo Store ID, previously referred to as `app key`. Follow Yotpo Loyalty's docs for [steps to retrieve this information](https://support.yotpo.com/docs/finding-your-yotpo-app-key-and-secret-key) |
| **Secret key**                 | Your Yotpo secret key. Follow Yotpo Loyalty's docs for [steps to retrieve this information](https://support.yotpo.com/docs/finding-your-yotpo-app-key-and-secret-key) |




## Integrating with Yotpo Loyalty
To integrate an existing system with Yotpo Loyalty:

1. Navigate to **Data map** â†’ **View Systems** and choose the system that you want to connect.

3. Pick the vendor or system type that you want to integrate with from the **Connection type** drop-down menu.
4. Complete the required fields for the integration and click **Save**. 

You may confirm your connection is working using the **Test connection** button, if desired.

To learn more, please see our guide for [Managing Integrations](/user-guides/integrations/manage-integrations).

## Technical detail 

Fides utilizes _API endpoints_ to access the _API service_ for a SaaS application. An _API service_ is the programmatic interface through which Fides can query and update data within an application to access, rectify, or delete personal information. Each SaaS tool will have a unique set of endpoints to enable privacy functions. 

### Yotpo Loyalty endpoints
In the table below, you can find which endpoints are used for each type of privacy function. Clicking on the endpoint will take you to the documentation for that endpoint.

| Privacy function | Endpoint | Description |
|----|----|----|
| **Access requests**            | [Customer](https://loyaltyapi.yotpo.com/reference/fetch-customer-details) | Retrieves customer information |

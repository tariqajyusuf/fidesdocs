# Yotpo Loyalty
[Yotpo Loyalty](https://www.yotpo.com/products/loyalty-and-referrals/) is a customer loyalty and rewards platform that enables businesses to create and manage loyalty programs.

## Prerequisites

In order to [integrate](../) with Yotpo Loyalty, you'll need to collect the following information:

| Name                       | Description                                                                                                                             |
| :------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **Domain***                | Your Yotpo Loyalty URL (default: `api.domo.com`)  |
| **Client ID***             | Your Yotpo Loyalty client ID. Follow Yotpo Loyalty's docs to retrive your [client ID and secret](https://developer.domo.com/portal/8ba9aedad3679-ap-is#quickstart) |
| **Client Secret***         | Your Yotpo Loyalty client secret. Follow Yotpo Loyalty's docs to retrieve your [client ID and secret](https://developer.domo.com/portal/8ba9aedad3679-ap-is#quickstart)|

## Integrating with Yotpo Loyalty
To integrate an existing system with Yotpo Loyalty:

1. Navigate to **Data map** â†’ **View Systems** and choose the system that you want to connect.

## Technical detail 

Fides utilizes _API endpoints_ to access the _API service_ for a SaaS application. An _API service_ is the programmatic interface through which Fides can query and update data within an application to access, rectify, or delete personal information. Each SaaS tool will have a unique set of endpoints to enable privacy functions. 

### Yotpo Loyalty endpoints
In the table below, you can find which endpoints are used for each type of privacy function. Clicking on the endpoint will take you to the documentation for that endpoint.

| Privacy function | Endpoint | Description |
|----|----|----|
| **Access Requests** | [User Read](https://developer.domo.com/portal/7269956a660d7-list-users) | Used to interact with the Yotpo Loyalty user information. |
| **Update or Delete Requests** | [User Update](https://developer.domo.com/portal/6d07be3b66fad-update-a-user) | Used to mask information about a user |

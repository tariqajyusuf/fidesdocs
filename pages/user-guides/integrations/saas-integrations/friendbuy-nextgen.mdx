# Friendbuy Nextgen

<Callout emoji=""> This Integration requires Fides Cloud or Fides Enterprise. For more information, [talk to our solutions team.](https://ethyca.com/book-demo?utm_source=docs-site) </Callout>


[Friendbuy](https://www.friendbuy.com) (Nextgen) is a customer referral platform that helps businesses create and manage referral programs to acquire new customers through word-of-mouth marketing.

## Prerequisites

In order to [integrate](/user-guides/integrations) with Friendbuy Nextgen, you'll need to collect the following information:

| Name         | Description                                                                                                                                                                                               |
| :----------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Domain\*** | The Friendbuy Nextgen API domain (default: mapi.fbot.me/v1)                                                                                                                                                      |
| **Key\***    | Your API key. Contact Friendbuy support to [obtain your key](https://developers.friendbuy.com/#authorization)       |
| **Secret\*** | Your API secret. Contact Friendbuy support to [obtain your secret](https://developers.friendbuy.com/#authorization) |

## Integrating with Friendbuy Nextgen

To integrate an existing system with Friendbuy Nextgen:

1. Navigate to **Data map** â†’ **View Systems** and choose the system that you want to connect.

## Technical detail

Fides utilizes _API endpoints_ to access the _API service_ for a SaaS application. An _API service_ is the programmatic interface through which Fides can query and update data within an application to access, rectify, or delete personal information. Each SaaS tool will have a unique set of endpoints to enable privacy functions.

### Friendbuy Nextgen Endpoints

In the table below, you can find which endpoints are used for each type of privacy function. Clicking on the endpoint will take you to the documentation for that endpoint.

| Privacy function              | Endpoint                                                                                                                                                                         | Description                               |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------- |
| **Access Requests**           | [Request User Data](https://developers.friendbuy.com/?hsCtaTracking=cac32c73-eaac-435c-aa4a-c906e6a3d3aa%7Cd9587bfd-7134-4f5b-ab33-53fdcf88fe23#request-user-data-for-ccpa-gdpr) | Returns information about a user.         |
| **Update or Delete Requests** | [Delete User Data](https://developers.friendbuy.com/?hsCtaTracking=cac32c73-eaac-435c-aa4a-c906e6a3d3aa%7Cd9587bfd-7134-4f5b-ab33-53fdcf88fe23#deleting-user-data-for-ccpa-gdpr) | Used to mask or erase records for a user. |

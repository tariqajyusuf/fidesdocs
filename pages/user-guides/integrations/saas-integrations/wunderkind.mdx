# Wunderkind
[Wunderkind](https://www.wunderkind.co/) is a marketing analytics platform specializing in personalized messaging solutions.

## Prerequisites

In order to [integrate](/docs/user-guides/integrations) with Wunderkind, you'll need to collect the following information:

| Name                       | Description                                                                                                                             |
| :------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **Website ID***            | Your Wunderkind website ID.                                                                                                            |

## Integrating with Wunderkind
To integrate an existing system with Wunderkind:

1. Navigate to **Data map** â†’ **View Systems** and choose the system that you want to connect.
2. Click on the **Integrations** tab.
3. Pick the vendor or system type that you want to integrate with from the **Connection type** drop-down menu.

## Technical detail

Fides utilizes _API endpoints_ to access the _API service_ for a SaaS application. An _API service_ is the programmatic interface through which Fides can query and update data within an application to access, rectify, or delete personal information. Each SaaS tool will have a unique set of endpoints to enable privacy functions.

### Wunderkind endpoints
In the table below, you can find which endpoints are used for each type of privacy function.

| Privacy function | Endpoint | Description |
|----|----|----|
| **Consent Requests** | `Opt In` | Used to process consumer opt-ins for the sale of personal information |
| **Consent Requests** | `Opt Out` | Used to process consumer opt-outs for the sale of personal information |

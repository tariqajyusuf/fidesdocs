# Unbounce
[Unbounce](https://unbounce.com//) is a landing page builder and conversion optimization platform that allows marketers to create, test, and optimize high-converting landing pages for their online marketing campaigns.

## Prerequisites

In order to [integrate](../) with Unbounce, you'll need to collect the following information:

| Name                       | Description                                                                                                                             |
| :------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **Domain***                | Your Unbounce URL (default: `api.unbounce.com`)  |
| **Client ID***             | Your Unbounce client ID. Follow Unbounce's docs to retrive your [client ID and secret](https://developer.domo.com/portal/8ba9aedad3679-ap-is#quickstart) |
| **Client Secret***         | Your Unbounce client secret. Follow Unbounce's docs to retrieve your [client ID and secret](https://developer.domo.com/portal/8ba9aedad3679-ap-is#quickstart)|
| **Redirect URL*** | The URL to which Fides will be redirected after successful authentication. | 

## Integrating with Unbounce
To integrate an existing system with Unbounce:

1. Navigate to **Data map** â†’ **View Systems** and choose the system that you want to connect.
2. Click on the **Integrations** tab.
3. Pick the vendor or system type that you want to integrate with from the **Connection type** drop-down menu.
4. Complete the required fields for the integration and click **Save**. 

You may confirm your connection is working using the **Test connection** button, if desired.

To learn more, please see our guide for [Managing Integrations](/user-guides/integrations/manage-integrations).

## Technical detail 

Fides utilizes _API endpoints_ to access the _API service_ for a SaaS application. An _API service_ is the programmatic interface through which Fides can query and update data within an application to access, rectify, or delete personal information. Each SaaS tool will have a unique set of endpoints to enable privacy functions. 

### Unbounce endpoints
In the table below, you can find which endpoints are used for each type of privacy function. Clicking on the endpoint will take you to the documentation for that endpoint.

| Privacy function | Endpoint | Description |
|----|----|----|
| **Access Requests** | [Pages](https://developer.unbounce.com/api_reference/#id_pages) | Returns a list of all pages for a given principal. |
| **Access Requests** | [Leads](https://developer.unbounce.com/api_reference/#id_pages__page_id__leads) | Returns a list of all leads for a given page. |
| **Erasure Requests** | [Lead delete](https://developer.unbounce.com/api_reference/#id_pages__page_id__lead_deletion_request) | Used to erase leads |

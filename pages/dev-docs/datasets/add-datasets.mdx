# Uploading datasets 

In order to use datasets for processing privacy requests, you must first add, or upload them to Fides.

## Adding and uploading datasets to Fides

You can upload or add your datasets to Fides via the Control administrative UI or Fides CLI:
* [Upload a dataset in Fides Control administrative UI](/dev-docs/datasets/add-datasets/add-datasets-ui)
* [Push a dataset using Fides CLI](/dev-docs/datasets/add-datasets/add-datasets-cli)

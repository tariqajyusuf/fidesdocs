# Command: view

The ```fides view``` command prints the current configuration of the Fides instance.

## Usage
Usage: ```fides view [args]```

This command prints the specified resource. Today this only supports the following options:

* <span class="nx-absolute -nx-mt-20" id="config"></span>[```config```](#config) - Print the current configuration of the Fides instance.

Running this command should result in output that resembles the following example. Note, depending on the configuration of your Fides instance this may contain a much longer list of configuration variables.
To learn more about all Fides configuration variables read the [Fides Configuration Variables](/dev-docs/configuration/configuration) guide.

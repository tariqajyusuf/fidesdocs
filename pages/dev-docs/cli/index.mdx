# Fides CLI Documentation

Learn Fides' CLI-based workflows. You can use the CLI to interact with the Fides Server.

> **Hands-on:** Try the [Fides: Get Started](#LINK) tutorials.

This is the documentation for Fides CLI. It is relevant to anyone working with Fides' CLI-based workflows; this is suitable for teams using both Fides Open-Source or Fides Enterprise.

Notably, this documentation does not cover the syntax and usage of the Fides language. For that, see the [Fides Language Documentation](#LINK).

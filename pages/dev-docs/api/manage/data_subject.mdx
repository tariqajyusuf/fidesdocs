# API: data_subject

The `/api/v1/data_subject` routes are used to to create, read, update, and delete the [Data Subjects](/tutorials/data-mapping/data-mapping-glossary#data-subject) within the Fides platform. Data Subjects can be customized but we recommend you use the [Fideslang](https://ethyca.github.io/fideslang/explorer/) taxonomy.
